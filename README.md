# Chess Problems API

Esta es una API para gestionar problemas de ajedrez. Incluye funcionalidades como:
- Registro e inicio de sesión.
- Gestión de problemas de ajedrez.
- Filtrado por temas y dificultad.

## Tecnologías
- Nest.js (TypeScript)
- MongoDB (Mongoose)
- JWT para autenticación
